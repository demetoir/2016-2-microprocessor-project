
# Fridge Lock - 마이크로프로세서를 이용한 OTP 보안 개별 잠금 냉장고 개발 프로젝트

-   OTP(One Time Pass) 보안 개별 장금 냉장고
-   웹페이지 상에서 원하는 냉장고 칸을 대여 및 반납
-   대여한 냉장고 칸은 OTP 비밀번호를 사용하여 잠금 및 잠금 해제
-   ARM cortex m3 마이크로프로세서 기반
-   Flask(python)를 이용한 웹서버 구성
